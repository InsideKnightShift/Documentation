OBJECTS

Object Tab enables the inclusion of various objects into the map e.g. units, trees, additional elements.

This Tab includes lists of grouped objects:

  • special – this group includes – players’ starting points, markers, sounds, special markers for RPG mode – see chapter: RPG mode map;
  • active objects – buildings, characters, special characters, artefacts;
  • neutral objects– holy places, entrances, wooden and stone buildings, marble structures , castles, stone buildings, traps, floors, bridges;
  • passives – decorative items – trees, plants, additional supplements, barrels, torches.
In order to place an object into a map, you first need to select it from the list and then click in the camera view. By pressing the Enter key on the numerical keypad, you can rotate the object in any direction before it is placed.

Attention: If the object is highlighted in green, you have chosen a correct possibility for placing it. Red is a warning that it is not possible to to place an object in a chosen location for one of many possible reasons: inappropriate terrain, water or another object has already been put in there.

There is a list of objects that have been recently used. By clicking on one, you can once again place it on the map.

Starting points

After clicking on „Specials”/”starting points” , starting points are shown for all players.

By using arrows you can move in within the list as well as add new players. In order to add a new player you must select the last one from the list and then click on the right arrow. Finally, you need to place the new player’s starting point into the map.

Starting points are the positions on the map from which the players begin their game. Placing starting points onto a map is required for subsequent map saving. Starting points for the first two players are set automatically when a new map is being drawn.. For other players, starting points must be placed manually.

Attention:

  • when placing a starting point you need to remember to put a piece of road next to it so the player is able to construct buildings.

  • there must also be pasture land created nearby

  • starting points cannot be placed on sand, rocks or hilly terrain.
