SOUNDS

There is a set of sound markers in Object Tap/Specials. They can be put onto a map in order to gain a realistic sound background. E.g. water roar marker can be placed on a river. To achieve the best effect, markers should be placed every five fields forming a line running down the middle of the river.

Now, when everything has been done all we have to do is to save our map. Choose the Save option from the Menu, fill in the map name and you are ready to start playing.
