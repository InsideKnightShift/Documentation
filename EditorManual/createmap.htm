CREATE A NEW MAP

A dialogue window pops up after you have clicked on Menu and chosen the "New Map" option. You can set all the basic parameters in it.

The dialogue window includes the following options:


· Map name that will be used in the game.
· File name which the map is to be written into, on your hard disk.
· Map size, from the smallest (96x96) up to the largest (256x256)
· Terrain type
