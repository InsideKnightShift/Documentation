GENERATOR

With the Generator Tab it is possible to pick one of the listed terrains randomly. 

It can also automatically fill the terrain with textures

Generate type 1. randomly generates a terrain ( algorithm 1)

Generate type 2. randomly generates a terrain ( algorithm 2)

Generate noise: adds noise at random

Add to altitudes: adds a new altitude map to the existing  one.

Invert altitudes: inverts altitudes on a map

Generate textures: automatically applies textures to a map.

Undo: reverses the last action
